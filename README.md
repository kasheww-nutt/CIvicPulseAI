# CivicPulse AI

**Community Hero - Hyperlocal Problem Solver**

CivicPulse AI is a mobile-first civic field-reporting application designed to help citizens identify, verify, and resolve local civic issues. It transforms reporting from a tedious form-filling exercise into a fast, photo-first experience.

## Core Features
*   **Photo-First Reporting**: Snap a photo of a civic issue. CivicPulse AI will extract the category, severity, and location details.
*   **Hyperlocal Missions**: View urgent nearby cases and verify them to earn Civic Trust.
*   **Civic Trust System**: Build your reputation as a reliable citizen by verifying reports and confirming fixes.
*   **Duplicate Fusion**: Identifies similar reports to prevent map clutter and prioritize municipal response.
*   **Reviewer Dashboard**: A dedicated view for civic reviewers to process authority-ready packets.

## Tech Stack
*   React, Vite, Tailwind CSS
*   Google Maps Platform (planned)
*   Gemini API (planned)

## Local Development
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.

*Note: The current municipal submission and export functionality is a demo (prepared for review) and is not automatically submitted to a real authority.*
